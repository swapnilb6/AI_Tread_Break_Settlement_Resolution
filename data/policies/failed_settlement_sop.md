# Purpose
This SOP explains how operations analysts should investigate and resolve failed settlement cases.

# Initial Validation
Validate trade attributes, settlement date, SSI linkage, and counterparty setup before taking any action.

# Root Cause Checks
Review whether the exception is caused by SSI mismatch, missing instruction, wrong account, booking mismatch, or holiday impact.

# Escalation Rules
If evidence is weak, contradictory, or suggests booking correction or SSI change, route the case for human approval.

# Audit Requirements
Record evidence sources, timestamps, decision rationale, and final action in the audit trail.
